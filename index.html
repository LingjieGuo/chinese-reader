工具栏图标 · Toolbar Icons
显示拼音 · Pinyin
一键显示或隐藏阅览区所有汉字上方的拼音注音。
Toggle all pinyin annotations above characters on or off.
高亮开关 · Highlight
开启后点击汉字即可标记或取消彩色高亮；图标旁的色块用于切换黄、红、绿、蓝四种颜色。
Enable highlight mode, then click any character to apply or remove a color mark. Use the color swatches to switch between yellow, red, green, and blue.
编辑内容 · Edit Content
在阅览区直接修改或粘贴新的中文内容。点击"完成编辑"后，拼音与排版将自动刷新。
Edit or paste new Chinese text directly in the reading area. Click "Done Editing" to regenerate pinyin and layout automatically.
汉字字号 · Character Size
弹出滑轨，拖动或直接输入数值来调整汉字的显示大小（18–64 px）。
Open a slider to drag or type a value to adjust character size (18–64 px).
拼音字号 · Pinyin Size
单独调整拼音注音的字号大小（10–32 px），不影响汉字大小。
Independently resize the pinyin annotations (10–32 px) without affecting the characters.
打印页面 · Print
将当前阅读视图（含拼音和高亮）直接打印，或通过浏览器另存为 PDF。
Print the reading view with pinyin and highlights, or save as PDF via your browser's print dialog.
阅览区操作 · Reader Interactions
  1. 点击汉字 · Tap a character
    在拼音模式开启时，单击某个汉字可单独显示或隐藏该字的拼音，不影响其他字。
    With Pinyin on, tap any character to show or hide only that character's pinyin, leaving others unchanged.
  2. 点击拼音 · Tap a pinyin
    直接点击某字上方的拼音文字，即可进入手动编辑状态——适合修正多音字或专有名词的读音。按 Enter 或点击别处确认，按 Esc 取消。
    Tap any pinyin text to edit it inline — great for polyphonic characters or proper nouns. Press Enter or click elsewhere to confirm; press Esc to cancel.
  3. 高亮模式下点击 · Tap in Highlight mode
    打开高亮开关后，点击汉字即添加所选颜色的高亮；再次点击同一汉字则取消高亮。
    With Highlight enabled, tap a character to apply the selected color; tap the same character again to remove it.
